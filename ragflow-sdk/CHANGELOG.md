# RagFlow SDK 更新日志

## [0.4.0] - 2024-11-07

### 变更
- 重构：从SDK核心代码中移除对Node.js `fs`模块的直接依赖
- 修改：`createFormData`函数现在支持Buffer和对象类型参数
- 更新：文档上传API接口支持多种文件输入类型
- 优化：将文件系统操作限制在测试代码中

### 说明
此版本提高了SDK的兼容性，使其核心代码不再直接依赖Node.js特有的`fs`模块，增强了在不同JavaScript环境中的适用性。
